               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>I run a loop on documents to discover what fields are searchable. <br> UnStored fields return false when I call isIndexed(). Converting these <br> fields to Text fields in the indexer corrects the problems.</p> 
                  <p>Obviously, using Text fields is a workaround, but I think the behaviour is <br> incorrect.</p> 
                  <p>I've reproduced the issue on MacOS X and Linux. I'm using rc4.</p>
