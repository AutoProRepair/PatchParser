                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>With code like the following: </p> 
                  <p> DatabaseMetaData dmd = conn.getMetaData(); <br> ResultSet rs = dmd.getBestRowIdentifier(null,"APP","a",3,true);<br> ResultSetMetaData rsmd = rs.getMetaData(); <br> int actualCols = rsmd.getColumnCount(); <br> for (int i = 0; i &lt; actualCols; i++) </p> { System.out.print("getColumnName: " + rsmd.getColumnName(i+1) + ", isNullable: "); System.out.println(rsmd.isNullable(i+1)); } 
                  <p>The printed values for isNullable returned are opposite of what they are when the getBestRowIdentifier call looks like this:<br> ResultSet rs = dmd.getBestRowIdentifier(null,"APP","a",1,true);</p> 
                  <p>In the latter case, the values are:<br> getColumnName: SCOPE, isNullable: 0<br> getColumnName: COLUMN_NAME, isNullable: 1<br> getColumnName: DATA_TYPE, isNullable: 0<br> getColumnName: TYPE_NAME, isNullable: 1<br> getColumnName: COLUMN_SIZE, isNullable: 0<br> getColumnName: BUFFER_LENGTH, isNullable: 0<br> getColumnName: DECIMAL_DIGITS, isNullable: 0<br> getColumnName: PSEUDO_COLUMN, isNullable: 0</p> 
                  <p>In the first case, the values are:<br> getColumnName: SCOPE, isNullable: 1<br> getColumnName: COLUMN_NAME, isNullable: 0<br> getColumnName: DATA_TYPE, isNullable: 1<br> getColumnName: TYPE_NAME, isNullable: 1<br> getColumnName: COLUMN_SIZE, isNullable: 1<br> getColumnName: BUFFER_LENGTH, isNullable: 1<br> getColumnName: DECIMAL_DIGITS, isNullable: 1<br> getColumnName: PSEUDO_COLUMN, isNullable: 1</p> 
                  <p>The isNullable value should be stable. <br> It's probably worthwhile verifying what the value <b>should</b> be in the first place.</p> 
                 </div> 
                </div> 
               </div>
              </div>
