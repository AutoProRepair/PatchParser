               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>original title: JSON-Specific Parameters arrntv causing some error for spellcheck component</p> 
                  <p>So I tried using the new array named list arrntv from solr 6.5 jenkins build<br> but the json returned was broken when it returned response for spellcheck with word break.<br> for example :</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-javascript"> {<span class="code-quote">"name"</span>:<span class="code-quote">"collation"</span>,{
        <span class="code-quote">"type"</span>:<span class="code-quote">"str"</span>,<span class="code-quote">"value"</span>:<span class="code-quote">"collationQuery"</span>:<span class="code-quote">"indomie kuing"</span>,
        <span class="code-quote">"hits"</span>:81,
        <span class="code-quote">"misspellingsAndCorrections"</span>:
        [
          {<span class="code-quote">"name"</span>:<span class="code-quote">"indomee"</span>,<span class="code-quote">"type"</span>:<span class="code-quote">"str"</span>,<span class="code-quote">"value"</span>:<span class="code-quote">"indomie"</span>},
          {<span class="code-quote">"name"</span>:<span class="code-quote">"kuih"</span>,<span class="code-quote">"type"</span>:<span class="code-quote">"str"</span>,<span class="code-quote">"value"</span>:<span class="code-quote">"kuing"</span>}
        ]}
 }
</pre> 
                   </div>
                  </div> 
                  <p>as you may see that "collationQuery":"indomie kuing" was considered as value thus causing the json to fail.<br> i think the correct json was :</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-javascript">{<span class="code-quote">"name"</span>:<span class="code-quote">"collation"</span>,
        <span class="code-quote">"type"</span>:<span class="code-quote">"object"</span>,
        <span class="code-quote">"value"</span>:{
        <span class="code-quote">"collationQuery"</span>:<span class="code-quote">"indomie kuing"</span>,
        <span class="code-quote">"hits"</span>:81,
        <span class="code-quote">"misspellingsAndCorrections"</span>:
        [
          {<span class="code-quote">"name"</span>:<span class="code-quote">"indomee"</span>,<span class="code-quote">"type"</span>:<span class="code-quote">"str"</span>,<span class="code-quote">"value"</span>:<span class="code-quote">"indomie"</span>},
          {<span class="code-quote">"name"</span>:<span class="code-quote">"kuih"</span>,<span class="code-quote">"type"</span>:<span class="code-quote">"str"</span>,<span class="code-quote">"value"</span>:<span class="code-quote">"kuing"</span>}
        ]}
 }
</pre> 
                   </div>
                  </div> 
                  <p>sorry for bad grammar english was not my first language and i know that object was not supported by current arrntv options.</p> 
                 </div>
