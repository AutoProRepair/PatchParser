               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>Attached 2 new classes:</p> 
                  <p>1) CoordConstrainedBooleanQuery<br> A boolean query that only matches if a specified number of the contained clauses<br> match. An example use might be a query that returns a list of books where ANY 2<br> people from a list of people were co-authors, eg:<br> "Lucene In Action" would match ("Erik Hatcher" "Otis Gospodnetić" "Mark Harwood"<br> "Doug Cutting") with a minRequiredOverlap of 2 because Otis and Erik wrote that.<br> The book "Java Development with Ant" would not match because only 1 element in<br> the list (Erik) was selected.</p> 
                  <p>2) CustomQueryParserExample<br> A customised QueryParser that allows definition of<br> CoordConstrainedBooleanQueries. The solution (mis)uses fieldnames to pass<br> parameters to the custom query.</p> 
                 </div>
