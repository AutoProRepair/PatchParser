                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>If no replicas of a shard are up and running, a search request gives the following response:</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">{
  <span class="code-quote">"responseHeader"</span>: {
    <span class="code-quote">"status"</span>: 503,
    <span class="code-quote">"QTime"</span>: 2,
    <span class="code-quote">"params"</span>: {
      <span class="code-quote">"q"</span>: <span class="code-quote">"*:*"</span>,
      <span class="code-quote">"indent"</span>: <span class="code-quote">"<span class="code-keyword">true</span>"</span>,
      <span class="code-quote">"wt"</span>: <span class="code-quote">"json"</span>,
      <span class="code-quote">"_"</span>: <span class="code-quote">"1432048084930"</span>
    }
  },
  <span class="code-quote">"error"</span>: {
    <span class="code-quote">"msg"</span>: <span class="code-quote">"no servers hosting shard: "</span>,
    <span class="code-quote">"code"</span>: 503
  }
}
</pre> 
                   </div>
                  </div> 
                  <p>The message should mention the shard which is down/unreachable.</p> 
                 </div> 
                </div>
