                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>A trigger action statement, such as an INSERT statement is not recompiled when there is some DDL change on the underlying table, such as a CREATE INDEX.</p> 
                  <p>In the example below a unique index is added to the table (t2) inserted into by the trigger's action statement. When the tirgger fires it does not raise any error (should raise a unique constraint violated error) and does not insert the value into the index. A select from t2 does not show the additional rows in t2 as it is performing an index scan, once the index is dropped the rows appear to the select.</p> 
                  <p>ij version 10.2<br> ij&gt; connect 'jdbc:derby:cs;create=true';<br> ij&gt; create table t (i int);<br> 0 rows inserted/updated/deleted<br> ij&gt; create table t2 (i int);<br> 0 rows inserted/updated/deleted<br> ij&gt; create trigger tt after insert on t for each statement mode db2sql<br> insert into t2 values 1;<br> 0 rows inserted/updated/deleted<br> ij&gt; insert into t values 1;<br> 1 row inserted/updated/deleted<br> ij&gt; select * from t2;<br> I<br> -----------<br> 1</p> 
                  <p>1 row selected<br> ij&gt; create unique index tu on t2<img class="emoticon" src="/jira/images/icons/emoticons/information.png" height="16" width="16" align="absmiddle" alt="" border="0">;<br> 0 rows inserted/updated/deleted<br> ij&gt; insert into t values 1;<br> 1 row inserted/updated/deleted<br> ij&gt; select * from t2;<br> I<br> -----------<br> 1</p> 
                  <p>1 row selected<br> ij&gt; insert into t values 1;<br> 1 row inserted/updated/deleted<br> ij&gt; select * from t2;<br> I<br> -----------<br> 1</p> 
                  <p>1 row selected<br> ij&gt; drop index tu;<br> 0 rows inserted/updated/deleted<br> ij&gt; select * from t2;<br> I<br> -----------<br> 1<br> 1<br> 1</p> 
                  <p>3 rows selected</p> 
                 </div> 
                </div> 
               </div>
              </div>
