                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>When parsing the Ruby output returned from Solr, if a numerical value has no value in the index, it causes an invalid Ruby hash to be returned. For instance:</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-xml"> 
 <span class="code-quote">'response'</span>=&gt;{<span class="code-quote">'numFound'</span>=&gt;1,<span class="code-quote">'start'</span>=&gt;0,<span class="code-quote">'maxScore'</span>=&gt;4.951244,<span class="code-quote">'docs'</span>=&gt;[
	{
	 <span class="code-quote">'subclass_t'</span>=&gt;<span class="code-quote">'Protocol'</span>,
	 <span class="code-quote">'pk_i'</span>=&gt;1,
	 <span class="code-quote">'id'</span>=&gt;<span class="code-quote">'Protocol:1'</span>,
	 <span class="code-quote">'name_t'</span>=&gt;<span class="code-quote">'Falcipain IC50'</span>,
	 <span class="code-quote">'group_id_i'</span>=&gt;,
	 <span class="code-quote">'score'</span>=&gt;4.951244}]
 }}
</pre> 
                   </div>
                  </div> 
                  <p>is not a valid hash because 'group_id_i' does not resolve to anything. It should resolve to nil:</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-xml"> 
 <span class="code-quote">'response'</span>=&gt;{<span class="code-quote">'numFound'</span>=&gt;1,<span class="code-quote">'start'</span>=&gt;0,<span class="code-quote">'maxScore'</span>=&gt;4.951244,<span class="code-quote">'docs'</span>=&gt;[
	{
	 <span class="code-quote">'subclass_t'</span>=&gt;<span class="code-quote">'Protocol'</span>,
	 <span class="code-quote">'pk_i'</span>=&gt;1,
	 <span class="code-quote">'id'</span>=&gt;<span class="code-quote">'Protocol:1'</span>,
	 <span class="code-quote">'name_t'</span>=&gt;<span class="code-quote">'Falcipain IC50'</span>,
	 <span class="code-quote">'group_id_i'</span>=&gt;nil,
	 <span class="code-quote">'score'</span>=&gt;4.951244}]
 }}
</pre> 
                   </div>
                  </div> 
                 </div> 
                </div>
