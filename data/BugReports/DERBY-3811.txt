               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>The attached Java program does the following, using two concurrent threads for the last part.<br> a) Inserts a 1 MB Clob.<br> b) UpdateThread changes the Clob to a 0.5 MB Clob, then sleeps for 5 seconds before it performs a rollback.<br> c) SelectThread selects the Clob and calls Clob.length().</p> 
                  <p>Both threads/connections run with READ_UNCOMMITTED isolation and with autocommit off.</p>
