                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>A user on IRC was having trouble getting file descriptor counts and JVM memory usage in the admin UI, but it worked perfectly fine on another system. The problem system uses OpenJDK, the other one uses the Apple JDK. The user had tracked it down to an exception while trying to get open file descriptor info.</p> 
                  <p>Looking in the SystemInfoHandler.java file, I see a comment reference to com.sun.management.UnixOperatingSystemMXBean at the point where it is getting file descriptor info. A little extra searching turned up <a href="https://issues.apache.org/jira/browse/ZOOKEEPER-1579" title="Compile error of UnixOperationSystemMXBean with open JDK" class="issue-link" data-issue-key="ZOOKEEPER-1579">ZOOKEEPER-1579</a> which refers to <a href="https://issues.apache.org/jira/browse/HBASE-6945" title="Compilation errors when using non-Sun JDKs to build HBase-0.94" class="issue-link" data-issue-key="HBASE-6945"><del>HBASE-6945</del></a>, the same problem with OpenJDK.</p> 
                 </div>
