                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>On client getTimestamp on a TIME column will print date 1900-01-01 instead of the current date like the embedded driver.<br> To repro run the <a href="https://issues.apache.org/jira/browse/DERBY-877" title="zOS - with DerbyClient getDate(#) fails with IllegalArgumentException - unsupported date format - resultset.java" class="issue-link" data-issue-key="DERBY-877"><del>DERBY-877</del></a> repro without specifying a file.encoding</p> 
                  <p>java TestEnc derbynetclient<br> <span class="error">[snip]</span><br> COLUMN 2:TM TIME<br> getString: 16:27:35<br> getTimeStamp: 1900-01-01 16:27:35.0<br> getTime: 16:27:35<br> getDate Exception SQLSTATE:null (EXPECTED)</p> 
                  <p>With Embedded it prints the current date for getTimestamp<br> java TestEnc derby</p> 
                  <p>COLUMN 2:TM TIME<br> getString: 16:27:35<br> getTimeStamp: 2006-01-28 16:27:35.0<br> getTime: 16:27:35<br> getDate Exception SQLSTATE:22005 (EXPECTED)</p> 
                 </div>
