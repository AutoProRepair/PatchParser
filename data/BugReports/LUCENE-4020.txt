               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>This is because the source array can be/ is different for each system/ JVM. So this pick is not repeatable for example:</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">  /** 
   * Return a random Locale from the available locales on the system.
   */
  <span class="code-keyword">public</span> <span class="code-keyword">static</span> Locale randomLocale(Random random) {
    Locale locales[] = Locale.getAvailableLocales();
    <span class="code-keyword">return</span> locales[random.nextInt(locales.length)];
  }
</pre> 
                   </div>
                  </div> 
                  <p>I don't think there is much we can do to make it repeatable (other than maybe enforcing locale using system property).</p> 
                 </div>
