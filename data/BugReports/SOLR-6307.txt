                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>Try to remove an element in the string array with curl:</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">curl http:<span class="code-comment">//localhost:8080/update\?commit\=<span class="code-keyword">true</span> -H <span class="code-quote">'Content-type:application/json'</span> -d <span class="code-quote">'[{ <span class="code-quote">"attr_birth_year_is"</span>: { <span class="code-quote">"remove"</span>: [1960]},  <span class="code-quote">"id"</span>: 1098}]'</span>
</span>
curl http:<span class="code-comment">//localhost:8080/update\?commit\=<span class="code-keyword">true</span> -H <span class="code-quote">'Content-type:application/json'</span> -d <span class="code-quote">'[{<span class="code-quote">"reserved_on_dates_dts"</span>: {<span class="code-quote">"remove"</span>: [<span class="code-quote">"2014-02-12T12:00:00Z"</span>, <span class="code-quote">"2014-07-16T12:00:00Z"</span>, <span class="code-quote">"2014-02-15T12:00:00Z"</span>, <span class="code-quote">"2014-02-21T12:00:00Z"</span>]}, <span class="code-quote">"id"</span>: 1098}]'</span></span>
</pre> 
                   </div>
                  </div> 
                  <p>Neither of them works.</p> 
                  <p>The set and add operation for int array works. <br> The set, remove, and add operation for string array works</p> 
                 </div> 
                </div> 
               </div>
