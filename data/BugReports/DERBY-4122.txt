                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>This bug was found when doing soft upgrade testing from Derby version 10.4.2.0 to 10.5.1.0 (RC1)</p> 
                  <p>Steps followed are as follows.</p> 
                  <p>1. Run setEmbeddedCP.bat from version 10.4.2.0's bin folder<br> 2. In a test folder run ij<br> 3. create system/wombat database.<br> ij&gt; connect 'jdbc:derby:system/wombat;create=true';<br> 4. exit ij<br> 5. Copy the 10.5.1.0 derby jars (from lib folder) and the derbyTesting.jar from 10.4.2.0 to the test folder and set classpath with them (including junit and ORO)<br> 6. Run suites.All <br> java -Xmx512M -Xms512M -Dderby.tests.trace=true junit.textui.TestRunner org.apache.derbyTesting.functionTests.suites.All</p>
