                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>Was doing a demo of spatial search and ran into the following problems:</p> 
                  <ol> 
                   <li>Some blocks of code were not formatted properly, hence losing the <tt><b>:</b></tt> parameters. Example: <a href="https://lucene.apache.org/solr/guide/6_6/spatial-search.html#SpatialSearch-bbox" class="external-link" rel="nofollow">https://lucene.apache.org/solr/guide/6_6/spatial-search.html#SpatialSearch-bbox</a></li> 
                   <li>The query mentioned for geodist section didn't actually return the scores. Need to add an fl parameter there. <a href="https://lucene.apache.org/solr/guide/6_6/spatial-search.html#SpatialSearch-geodist" class="external-link" rel="nofollow">https://lucene.apache.org/solr/guide/6_6/spatial-search.html#SpatialSearch-geodist</a></li> 
                  </ol> 
                 </div>
