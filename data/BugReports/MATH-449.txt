               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>Currently there is no storeless version for computing the covariance. However, Pebay (2008) describes algorithms for on-line covariance computations, <a href="http://infoserve.sandia.gov/sand_doc/2008/086212.pdf" class="external-link" rel="nofollow">http://infoserve.sandia.gov/sand_doc/2008/086212.pdf</a>. I have provided a simple class for implementing this algorithm. It would be nice to have this integrated into org.apache.commons.math.stat.correlation.Covariance.</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java"><span class="code-comment">//This code is granted <span class="code-keyword">for</span> inclusion in the Apache Commons under the terms of the ASL.
</span>
<span class="code-keyword">public</span> <span class="code-keyword">class </span>StorelessCovariance{

    <span class="code-keyword">private</span> <span class="code-object">double</span> deltaX = 0.0;
    <span class="code-keyword">private</span> <span class="code-object">double</span> deltaY = 0.0;
    <span class="code-keyword">private</span> <span class="code-object">double</span> meanX = 0.0;
    <span class="code-keyword">private</span> <span class="code-object">double</span> meanY = 0.0;
    <span class="code-keyword">private</span> <span class="code-object">double</span> N=0;
    <span class="code-keyword">private</span> <span class="code-object">Double</span> covarianceNumerator=0.0;
    <span class="code-keyword">private</span> <span class="code-object">boolean</span> unbiased=<span class="code-keyword">true</span>;

    <span class="code-keyword">public</span> Covariance(<span class="code-object">boolean</span> unbiased){
	<span class="code-keyword">this</span>.unbiased = unbiased;
    }

    <span class="code-keyword">public</span> void increment(<span class="code-object">Double</span> x, <span class="code-object">Double</span> y){
        <span class="code-keyword">if</span>(x!=<span class="code-keyword">null</span> &amp; y!=<span class="code-keyword">null</span>){
            N++;
            deltaX = x - meanX;
            deltaY = y - meanY;
            meanX += deltaX/N;
            meanY += deltaY/N;
            covarianceNumerator += ((N-1.0)/N)*deltaX*deltaY;
        }
        
    }

    <span class="code-keyword">public</span> <span class="code-object">Double</span> getResult(){
        <span class="code-keyword">if</span>(unbiased){
            <span class="code-keyword">return</span> covarianceNumerator/(N-1.0);
        }<span class="code-keyword">else</span>{
            <span class="code-keyword">return</span> covarianceNumerator/N;
        }
    }
