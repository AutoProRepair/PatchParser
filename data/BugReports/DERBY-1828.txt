                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>The SQL standard says that SQL State '42' is for "syntax error or access<br> rule violation" (section 23.1).</p> 
                  <p>There is a question of what JDBC 4.0 exception should be thrown for a access rule violation,<br> JDBC 4.0 maps '42' to SQLSyntaxErrorException which seems wrong for an access rule.</p> 
                  <p>Message thread:</p> 
                  <p><a href="http://mail-archives.apache.org/mod_mbox/db-derby-dev/200609.mbox/%3c45004A04.5080002@apache.org%3e" class="external-link" rel="nofollow">http://mail-archives.apache.org/mod_mbox/db-derby-dev/200609.mbox/%3c45004A04.5080002@apache.org%3e</a></p> 
                 </div>
