                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p><b>example of committed change's usage:</b></p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">
{
  <span class="code-quote">"class"</span> : <span class="code-quote">"org.apache.solr.ltr.model.DefaultWrapperModel"</span>,
  <span class="code-quote">"name"</span> : <span class="code-quote">"myWrapperModelName"</span>,
  <span class="code-quote">"params"</span> : {
    <span class="code-quote">"resource"</span> : <span class="code-quote">"models/myModel.json"</span>
  }
}
</pre> 
                   </div>
                  </div> 
                  <hr> 
                  <p><b>original summary:</b><br> We add new model which contains only the location of the external model and loads it during the initialization.<br> By this procedure, large models which are difficult to upload to ZooKeeper can be available.</p> 
                  <p>The new model works as the wrapper of existing models, and deligates APIs to them.<br> We add two classes by this patch:</p> 
                  <ul> 
                   <li><tt>ExternalModel</tt> : a base class for models with external resources.</li> 
                   <li><tt>URIExternalModel</tt> : an implementation of <tt>ExternalModel</tt> which loads the external model from specified URI (ex. <a href="file:" class="external-link" rel="nofollow">file:</a>, http:, etc.).</li> 
                  </ul> 
                  <p>For example, if you have a model on the local disk "file:///var/models/myModel.json", the definition of <tt>URIExternalModel</tt> will be like the following.</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">
{
