                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>Derby 10.7 DatabaseMetaData.getTypeInfo() should not return the BOOLEAN data type with a soft upgraded database as often applications use getTypeInfo() to determine if tables can be created with this type.</p> 
                  <p>To reproduce and see the impact of the problem, first create the database testdb with 10.6<br> ij version 10.6<br> ij&gt; connect 'jdbc:derby:testdb;create=true';<br> ij&gt;</p> 
                  <p>run the attached program UseDBMetaForBool with 10.6 and it runs fine.<br> $ java UseDBMetaForBool<br> getDriverVersion10.6.2.3 - (1026030M)<br> supportsBoolean = false Make my table accordingly<br> CREATING SMALLINT TABLE SINCE NO BOOLEAN<br> getBoolean=true<br> getString=1</p>
