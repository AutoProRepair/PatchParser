                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>A query involving "&lt; ALL" does not yield the right results for decimal datatype. "&lt; ANY" works, though.</p> 
                  <p>To reproduce the issue:</p> 
                  <p>% create table t1(col1 decimal(10,5));<br> % insert into t1 values (-21483.64800);<br> % insert into t1 values (74837.00000);<br> % select col1 from t1 where col1 &lt; ALL (select 0.0 from t1);</p> 
                  <p>The above yields no results whereas it should return the first row.</p> 
                 </div> 
                </div>
