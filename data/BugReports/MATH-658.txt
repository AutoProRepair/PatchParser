               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>This issue concerns the FastMath class and its test class.</p> 
                  <p>(1) In the double pow(double, double) function, there are 2 identical "if" blocks. The second one can be suppressed.<br> if (y &lt; 0 &amp;&amp; y == yi &amp;&amp; (yi &amp; 1) == 1) </p> { return Double.NEGATIVE_INFINITY; } 
                  <p> // this block is never used -&gt; to be suppressed<br> if (y &lt; 0 &amp;&amp; y == yi &amp;&amp; (yi &amp; 1) == 1) </p> { return -0.0; }
                  <br> 
                  <br> if (y &gt; 0 &amp;&amp; y == yi &amp;&amp; (yi &amp; 1) == 1) { return -0.0; } 
                  <p>(2) To obtain better code coverage, we added some tests case in FastMathTest.java (see attached file)</p> 
                  <ul class="alternate" type="square"> 
                   <li>Added test for log1p</li> 
                   <li>Added tests in testPowSpecialCases()</li> 
                   <li>Added tests for a 100% coverage of acos().</li> 
                   <li>Added tests for a 100% coverage of asin().</li> 
                  </ul> 
                 </div>
