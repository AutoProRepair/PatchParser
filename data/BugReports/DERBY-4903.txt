                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>The plan exporter tool fails to produce well-formed output for the following query:</p> 
                  <p>ij&gt; select * from sysibm.sysdummy1 where ibmreqd &lt; 'Z';</p> 
                  <p>The generation of XML doesn't fail, but when I open the file in Firefox, I see this message:</p> 
                  <p>XML Parsing Error: not well-formed<br> Location: <a href="file:///tmp/plan.xml" class="external-link" rel="nofollow">file:///tmp/plan.xml</a><br> Line Number 9, Column 11:</p> 
                  <p>Operator: &lt;<br> ----------^</p> 
                  <p>HTML generation prints the following error, and produces an empty file:</p> 
                  <p>ERROR: 'The value of attribute "scan_qualifiers" associated with an element type "null" must not contain the '&lt;' character.'</p> 
                 </div> 
                </div> 
               </div>
