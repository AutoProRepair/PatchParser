                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>When using a boost query (bq) that contains a parentheses (like this example from the Relevancy Cookbook section of the wiki):</p> 
                  <div class="preformatted panel" style="border-width: 1px;">
                   <div class="preformattedContent panelContent"> 
                    <pre> ? defType = dismax 
    &amp; q = foo bar 
    &amp; bq = (*:* -xxx)^999 
</pre> 
                   </div>
                  </div> 
                  <p>You get the following error:</p> 
                  <p>org.apache.lucene.queryparser.classic.ParseException: Cannot parse '<del>xxx)': Encountered " ")" ") "" at line 1, column 12. Was expecting one of: &lt;EOF&gt; &lt;AND&gt; ... &lt;OR&gt; ... &lt;NOT&gt; ... "+" ... "</del>" ... &lt;BAREOPER&gt; ... "(" ... "*" ... "^" ... &lt;QUOTED&gt; ... &lt;TERM&gt; ... &lt;FUZZY_SLOP&gt; ... &lt;PREFIXTERM&gt; ... &lt;WILDTERM&gt; ... &lt;REGEXPTERM&gt; ... "[" ... "{" ... &lt;NUMBER&gt; ...</p> 
                 </div> 
                </div> 
               </div>
