                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>While running tests on Ipv6 machines using the 10.3 jars with the default security manager, I had the following findings/questions<br> I started the server like this java org.apache.derby.drda.NetworkServerControl start -h 2002:92a:8f7a:13:9:42:74:19<br> and the server started with the following command<br> Security manager installed using the Basic server security policy.<br> Apache Derby Network Server - 10.3.1.0 beta - (548006) started and ready to accept connections on port 1527 at 2007-06-25 23:44: 36.835 GMT</p> 
                  <p>So I think the server is using the default security manager. Then when I tried to get conenction though ij</p> 
                  <p>got the following error message<br> Access denied (java.net.SocketPermission <span class="error">[2002:92a:8f7a:13:9:42:73:218]</span>:34016 accept,resolve)<br> java.security.AccessControlException: Access denied (java.net.SocketPermission <span class="error">[2002:92a:8f7a:13:9:42:73:218]</span>:34016 accept,resolve) <br> at java.security.AccessController.checkPermission(AccessController.java:104)<br> at java.lang.SecurityManager.checkPermission(SecurityManager.java:547)<br> at java.lang.SecurityManager.checkAccept (SecurityManager.java:1172)<br> at java.net.ServerSocket.implAccept(ServerSocket.java:466)<br> at java.net.ServerSocket.accept(ServerSocket.java:433)<br> at org.apache.derby.impl.drda.ClientThread$1.run (Unknown Source)<br> at java.security.AccessController.doPrivileged(AccessController.java:242)<br> at org.apache.derby.impl.drda.ClientThread.run(Unknown Source)</p> 
                  <p>I had the derby.properties file like this</p> 
                  <p>derby.database.sqlAuthorization=true<br> derby.connection.requireAuthentication=true<br> derby.infolog.append=true<br> derby.authentication.provider=BUILTIN<br> derby.stream.error.logSeverityLevel=0</p> 
                  <p>#derby.language.logStatementText=true</p> 
                  <ol> 
                   <li>User's Definitions<br> derby.user.user2=pass2</li> 
                  </ol> 
                 </div> 
                </div>
