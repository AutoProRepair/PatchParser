                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>Network Server don't start in czech localized enviroment due missing key DRDA_MissingNetworkJar.S in file org/apache/derby/loc/drda/messages_cs.properties</p> 
                  <p>Exception:<br> Thu Jul 23 15:56:24 CEST 2015 : null<br> java.lang.NullPointerException<br> at org.apache.derby.iapi.tools.i18n.LocalizedResource.getTextMessage(Unknown Source)<br> at org.apache.derby.impl.drda.NetworkServerControlImpl.localizeMessage(Unknown Source)<br> at org.apache.derby.impl.drda.NetworkServerControlImpl.localizeMessage(Unknown Source)<br> at org.apache.derby.drda.NetworkServerControl.installSecurityManager(Unknown Source)<br> at org.apache.derby.drda.NetworkServerControl.main(Unknown Source)</p> 
                  <p>Missing key is called in file org.apache.derby.drda.NetworkServerControl.java at line 818.</p> 
                  <p>Reproduce under windows in console run:<br> set DERBY_OPTS=-Duser.language=cs<br> startNetworkServer.bat</p> 
                  <p>Workaround:<br> Run derby server under en localization.<br> Windows consola run:<br> set DERBY_OPTS=-Duser.language=en<br> startNetworkServer.bat</p> 
                 </div> 
                </div>
