               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>If someone submits a pull request, i think we should put it in changes.txt in some way similar to the jira issues:</p> 
                  <p>e.g. for a JIRA issue we do:</p> 
                  <div class="preformatted panel" style="border-width: 1px;">
                   <div class="preformattedContent panelContent"> 
                    <pre>* LUCENE-XXXX: Add FooBar.  (Joe Contributor via John Committer)
</pre> 
                   </div>
                  </div> 
                  <p>changes2html recognizes and expands these to jira issue links.</p> 
                  <p>so I think we should be able to do something like:</p> 
                  <div class="preformatted panel" style="border-width: 1px;">
                   <div class="preformattedContent panelContent"> 
                    <pre>* pull request #xxx: Add FooBar. (Joe Contributor via John Committer)
</pre> 
                   </div>
                  </div> 
                  <p>and have it link to the request, too.</p> 
                 </div>
