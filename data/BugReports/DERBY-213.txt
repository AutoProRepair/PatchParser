                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>Network Server closes the result set if ResultSet.next() is <br> called after the last row of the result set. The test code <br> below throws the following exception.</p> 
                  <p>SQLState: null<br> Severity: -99999<br> Message: Invalid operation: result set closed<br> com.ibm.db2.jcc.am.SqlException: Invalid operation: result set <br> closed<br> at <br> com.ibm.db2.jcc.am.ResultSet.checkForClosedResultSet(ResultSet.j<br> ava:3419)<br> at <br> com.ibm.db2.jcc.am.ResultSet.nextX(ResultSet.java:290)<br> at <br> com.ibm.db2.jcc.am.ResultSet.next(ResultSet.java:277)<br> at AfterLast.test(AfterLast.java:75)<br> at AfterLast.main(AfterLast.java:32)</p> 
                  <p>stmt.executeUpdate("CREATE TABLE TAB ( I INT)");<br> stmt.executeUpdate("INSERT INTO TAB VALUES(1)");<br> stmt.executeUpdate("INSERT INTO TAB VALUES(2)");<br> String sql ="SELECT * from tab"; <br> ps = conn.prepareStatement(sql);<br> ResultSet rs = ps.executeQuery();<br> System.out.println(sql);<br> while (rs.next())<br> System.out.println(rs.getInt(1));<br> try </p> { System.out.println("one more next"); rs.next(); } 
                  <p> catch (Exception e)</p> { System.out.println("FAIL: next should return false not throw exception"); e.printStackTrace(); } 
                 </div> 
                </div>
